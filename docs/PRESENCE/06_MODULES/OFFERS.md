# Module: Offers — **DEFERRED (Phase 3+)**

Coupons/discount-offer functionality requires consumer-side density across multiple businesses in the same micro-geo to be meaningful (cross-promotion between 5 businesses is meaningless; between 500 it's a moat — this business has neither at launch). Do not build without an explicit override conversation. If a customer requests this in Phase 1-2, the answer is "not yet, here's why" — not a quick feature bolt-on.
