# Feature List — Phased

## P0 (Phase 1, build now)
- GBP: read reviews, AI-drafted + templated responses, Q&A monitoring, listing accuracy alerts
- WhatsApp: review-request triggers, drip sequences, credit-metered broadcast
- Meta: read + basic scheduled posting (NOT ads management — that's Meta Ads' job)
- AI: review-response drafting, content generation, weekly insight summaries (cache-first)
- Attribution v0: GBP direction-requests + call-tracking + WhatsApp-campaign correlation (proxy signals)
- Agency console: multi-client switcher, white-label branding, consolidated reporting

## P1 (Phase 2)
- Premium analytics tier (deeper attribution/cohort views)
- Multi-location dashboard + role-based access (Scale tier)
- Attribution v1: cohort-level correlation dashboards

## DEFERRED (Phase 3+, do not build without explicit override)
- API/integration layer (Zoho, POS systems, WABA providers)
- Featured listing/ads
- Lead generation
- Marketplace (2-sided)
- Commission-based revenue / embedded payments beyond subscription billing
- Coupons / Loyalty network / Cross-promotion
- Franchise tooling
- Full multi-touch attribution / aggregate cohort benchmarking (Year 2+ if data volume justifies)
