# Business Model

## Revenue Architecture — Build, Sequence, Kill
Two primitives at launch. Everything else is a menu, not a Day-1 architecture — running all 13 models from a generic hyperlocal-SaaS brief simultaneously is how these companies die (fragmented focus, unclear retention lever).

| Model | Verdict |
|---|---|
| Tiered subscription | **BUILD — Day 1** |
| AI credits (metered) | **BUILD — Day 1** — the NRR engine |
| WhatsApp conversation credits | **BUILD — Day 1, pass-through priced** — never absorb into subscription margin |
| Premium analytics tier | Phase 1.5 (month 3–6) |
| Enterprise/multi-location | Phase 2 |
| White-label (agency reseller) | **Phase 1, PARALLEL TRACK — the actual GTM wedge, not an afterthought** |
| API access | Phase 2–3 |
| Featured listing/ads | Phase 3, conditional on real consumer MAU |
| Lead generation | Phase 3 |
| Marketplace (2-sided) | **DO NOT BUILD at MVP** — two-sided liquidity cold-start, proven hard in India (Justdial/Magicpin pattern) |
| Commission-based revenue | **DO NOT BUILD at MVP** — opens RBI PA/PG compliance surface pre-PMF |
| Coupons/Loyalty/Cross-promotion | Defer to Phase 3+ — requires consumer-side density this business won't have at launch |
| Franchise | Do not model in Year 1–2 |

## Unit Economics by Segment (validate against Phase 0 primary research before trusting)
| Segment | ACV/mo | CAC | Gross Margin | Est. Churn/mo | LTV:CAC |
|---|---|---|---|---|---|
| Cafes/QSR/Restaurants | ₹4,000–7,000 | ₹3,500–5,000 | ~70% | 6–8% | 4–6x |
| Salons/Spas/Gyms | ₹5,000–10,000 | ₹4,000–6,000 | ~72% | 4–6% | 6–9x |
| Clinics/Healthcare | ₹8,000–15,000 | ₹5,000–8,000 | ~75% | 3–5% | 9–14x |
| Retail/Fashion/Jewellery | ₹6,000–12,000 | ₹4,000–7,000 | ~68% | 5–7% | 6–9x |
| **Agency white-label (per agency)** | ₹25,000–75,000 | ₹15,000–30,000 (one-time) | ~80%+ | <2% | **15–30x** |

**The single most important number:** agency channel LTV:CAC crushes every direct-to-SMB row. This restructures GTM sequencing — direct sales prove the product and generate case studies; agency reselling is what actually justifies a ₹100 Cr ARR ambition.

**Biggest unvalidated variable:** F&B churn assumption. If real churn runs 8–10%/mo instead of 6–8%, that segment's LTV:CAC goes underwater. Phase 0 primary research must validate this before the model is trusted for fundraising.
